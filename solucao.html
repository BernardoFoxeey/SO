Solução phishing

O que é phishing?

Phishing é uma técnica de fraude online onde um atacante tenta enganar as vítimas para que revelem informações pessoais, como senhas e números de cartão de crédito, geralmente através de e-mails ou sites falsificados.

Como resolver?

Para resolver um ataque de phishing, é importante seguir algumas etapas:

  1. veirificar e orbservar o email é veridico, olhar a ortografia e a gramática
  2. Se o email for suspeito evite de clicar e notifique o suporte da empresa, ong ou marca e relata a situação
  3. Se você já clicou no link, não insira informações pessoais e altere suas senhas imediatamente
  4. Instale um software antivírus e mantenha-o atualizado
  5. olha a foto de perfil do email se corresponde ser o mesmo ao original

Solução Ransomware

O que é Ransomware?

Ransomware é um tipo de malware que criptografa os arquivos do usuário e exige um resgate para restaurar o acesso.

Como resolver?

  1. Desconecte o dispositivo da rede para evitar a propagação do malware
  2. Use um software antivírus para tentar remover o ransomware
  3. Restaure os arquivos a partir de backups, se disponíveis
  4. Considere entrar em contato com as autoridades, especialmente se os dados forem críticos
  5. Evite pagar o resgate, pois isso não garante a recuperação dos dados e incentiva os criminosos

Senhas Fracas

O que são senhas fracas?

São senhas que são facil de saber de saber como por exemplo "123456" ou "2468" ou combinações meio óbvios

Como resolver?

  1. Usa senhas mais fortes com caracteres especificos como @ _ e #
  2. Usa senhas longas com mais de 8 caracteres
  3. Usa senhas diferentes para cada conta

software Desatualizado

O que é software desatualizado?

Software desatualizado é aquele que não recebe mais atualizações de segurança, tornando-o vulnerável a ataques.

Como resolver?

  1. Smpre atualizar os programas e ferramentas pra evitar vírus
  2. Usar um software de atualização automática para garantir que todos os programas estejam sempre atualizados
  3. Desinstalar programas que não são mais utilizados
  4. Usar um software de segurança que possa detectar e alertar sobre software desatualizado
